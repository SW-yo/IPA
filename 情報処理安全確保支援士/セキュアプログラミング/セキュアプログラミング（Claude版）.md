## 📝 令和8年春 予想問題

### **問題1: 配列の要素数カウント**

**問題文**: 整数の配列arrと整数nが与えられる。配列arr内でn以上の値を持つ要素の個数を返す関数countを作成せよ。

```
○count(整数型の配列: arr, 整数型: n)
    整数型: cnt ← 0
    整数型: i
    for (i を 0 から arrの要素数-1 まで 1 ずつ増やす)
        if (arr[i] ≧ n)
            [  a  ]
        endif
    endfor
    return [  b  ]
○
```

**選択肢 [a]**:

1. `cnt ← cnt - 1`
2. `cnt ← cnt + 1`
3. `cnt ← arr[i]`
4. `i ← i + 1`

**選択肢 [b]**:

1. `cnt`
2. `i`
3. `n`
4. `arr[i]`

**解答**: [a]=2, [b]=1

**Java実装例**:

```java
public int count(int[] arr, int n) {
    int cnt = 0;
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] >= n) {
            cnt = cnt + 1;
        }
    }
    return cnt;
}
```

**JavaScript実装例**:

```javascript
function count(arr, n) {
    let cnt = 0;
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] >= n) {
            cnt = cnt + 1;
        }
    }
    return cnt;
}
```

### **問題2: 二分探索の実装**

**問題文**: 昇順にソートされた整数配列arrから、値targetを二分探索で見つけ、そのインデックスを返す。見つからない場合は-1を返せよ。

```
○binarySearch(整数型の配列: arr, 整数型: target)
    整数型: left ← 0
    整数型: right ← arrの要素数 - 1
    整数型: mid
    
    while (left ≦ right)
        mid ← [  c  ]
        if (arr[mid] = target)
            return mid
        elseif (arr[mid] < target)
            [  d  ]
        else
            [  e  ]
        endif
    endwhile
    return -1
○
```

**選択肢 [c]**:

1. `(left + right) ÷ 2`
2. `left + right`
3. `right - left`
4. `(right - left) ÷ 2`

**選択肢 [d]**:

1. `left ← mid + 1`
2. `right ← mid - 1`
3. `left ← mid - 1`
4. `right ← mid + 1`

**選択肢 [e]**:

1. `left ← mid + 1`
2. `right ← mid - 1`
3. `left ← mid`
4. `right ← mid`

**解答**: [c]=1, [d]=1, [e]=2

### **問題3: スタックを用いた括弧チェック**

**問題文**: 文字列strに含まれる括弧'('と')'が正しく対応しているかを判定する関数。正しければtrueを返す。

```
○checkBrackets(文字列型: str)
    スタック型: stack ← 新しいスタック
    整数型: i
    
    for (i を 0 から strの長さ-1 まで 1 ずつ増やす)
        if (str[i] = '(')
            [  f  ]
        elseif (str[i] = ')')
            if (stackが空)
                return false
            endif
            [  g  ]
        endif
    endfor
    return [  h  ]
○
```

**選択肢 [f]**:

1. `stackにpop`
2. `stackにpush '('`
3. `stackを初期化`
4. `何もしない`

**選択肢 [g]**:

1. `stackにpush ')'`
2. `stackからpop`
3. `stackを初期化`
4. `return true`

**選択肢 [h]**:

1. `true`
2. `false`
3. `stackが空`
4. `stackの要素数`

**解答**: [f]=2, [g]=2, [h]=3
